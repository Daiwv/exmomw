Для данной системы целостной логической единицей является сделка. 

Сделка представляет из себя два завершенных ордера: открытие (покупка) и закрытие (продажа)

Для сделки характерны такие параметры как stopLoss (ограничение потерь) и priceIntent (целевая стоимость, или ценовое намерение).

stopLoss 

Если курс (цена купленного актива) опускается до значения этого параметра, сделка должна быть завершена по этой цене.
В случае, если не удается завершить сделку по данной цене, ордер на продажу должен сопровождать (пересоздаваться) по самой высокой цене продажи до тех пор, пока сдедка не будет закрыта.
